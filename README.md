# codebase-skills

Codebase blog auto-posting skills.

## Install

Install `codebase-mcp` skill for selected agents:

```bash
npx skills add beyondeth/codebase-skills --skill codebase-mcp --agent codex claude-code gemini-cli antigravity -y
```

Install all skills from this repo for all supported agents:

```bash
npx skills add beyondeth/codebase-skills --all
```
